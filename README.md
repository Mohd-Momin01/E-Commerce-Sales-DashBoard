# E-Commerce-Sales-DashBoard
This project is an interactive E-Commerce Sales Dashboard built using Power BI. It helps in analyzing online sales data through different visualizations like bar charts, pie charts, maps, and slicers. Users can filter data, explore trends, and important insights The project focuses on making data easy to understand and useful for decision-making.
# E-Commerce Sales Dashboard (Power BI)

## About this project
I created this dashboard to understand and analyze online sales data in a simple way. The goal was to track sales, profit, and customer behavior using Power BI.

![Dashboard](./DashBoard%20Images/FULL%20INSIGHTS%20DASHBOARD.png)


## What this dashboard shows
- Total sales amount, profit, and average order value  
- Sales performance by state  
- Profit trends month-wise  
- Category-wise and sub-category-wise analysis  
- Customer-wise sales data  
- Payment mode distribution  

## Features
- Filters and slicers to explore data easily  
- Quarter and state selection for better analysis  
- Drill-down to check detailed information  
- Clean and interactive dashboard design  

## Data used
The dataset includes:
- Order details (Order ID, Date)  
- Customer information (Name, State, City)  
- Sales data (Amount, Profit, Quantity)  
- Product details (Category, Sub-category)  
- Payment mode  

## What I learned
- How to build interactive dashboards in Power BI  
- Data modeling and creating relationships  
- Using slicers and filters for better analysis  
- Presenting data in a clear and simple way
